Installation:

1. Make sure that Epic Games's MetaHumanForMaya plugin is already installed. You can check it's installation instructions at https://dev.epicgames.com/documentation/en-us/metahuman/installing-in-maya    
2. Copy both "MetaHumanExtraTools" folder and "MetaHumanExtraTools.mod" file into your Maya's module path (typically "C:\Users\UserName\Documents\maya\modules").
3. Restart Maya. Make sure that "MetaHumanExtraTools.py" is loaded in the plugin manager. You can find MetaHuman Extra Tools in the MetaHuman top menu.
